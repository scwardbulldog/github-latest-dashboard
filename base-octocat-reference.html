Color Palette

  • Body/Silhouette: #000 (Black)
  • Face: #F4CBB2 (Peach/Beige)
  • Eyes: #FFF (White)
  • Pupils: #AD5C51 (Reddish Brown)
  • Nose/Mouth: #AD5C51 (Reddish Brown)
  • Suction Cups: #C3E4D8 (Teal/Mint Green)

Dimensions

  • ViewBox: 0 0 379 334
  • Aspect Ratio: ~1.13:1 (width:height)
  • Recommended Display Sizes: 80px - 200px

Key SVG Elements

  1. #cat - Main body silhouette (black)
  2. #face - Facial skin area (peach)
  3. #eye-left, #eye-right - White of eyes
  4. #pupil-left, #pupil-right - Brown pupils
  5. #nose - Small circular nose
  6. #mouth - Curved smile path
  7. #octo1-9 - Tentacle suction cups (9 total)

Usage Guidelines

  • Can be scaled uniformly - use CSS transform: scale()
  • Can be rotated for dynamic poses
  • Maintain aspect ratio to avoid distortion
  • Keep color palette consistent for brand recognition
  • Can overlay accessories (skateboard, vehicles, etc.)

Scaling Examples

<!-- 50% smaller -->
<g transform="scale(0.5)">
    <!-- octocat paths -->
</g>

<!-- Rotated 15 degrees -->
<g transform="rotate(-15 189.5 167)">
    <!-- octocat paths -->
</g>

<!-- Combined: scaled and positioned -->
<g transform="translate(50, 20) scale(0.8)">
    <!-- octocat paths -->
</g>

Animation-Ready Grouping

For animations, wrap in semantic groups:

<g class="octocat-body">
    <!-- All cat/face/eye paths -->
</g>
<g class="tentacle-left">
    <!-- Left side suction cups -->
</g>
<g class="tentacle-right">
    <!-- Right side suction cups -->
</g>

Performance Notes

  • Total path complexity: Medium (acceptable for inline SVG)
  • Recommended: Use as inline SVG vs external file
  • Gzip-friendly (repetitive path commands compress well)
  • Animate with CSS transforms for GPU acceleration
