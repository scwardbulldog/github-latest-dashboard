Official GitHub Octocat Proportions

  • ViewBox: 0 0 379 334 - Official octocat dimensions
  • Body Color: #000 (Black) - The signature silhouette
  • Face Color: #F4CBB2 (Peach/Beige) - Warm face tone
  • Pupil Color: #AD5C51 (Reddish Brown) - Eye details
  • Suction Cups: #C3E4D8 (Teal/Mint) - Octopus tentacle details

SVG Path Sources

  • Base Octocat: Official GitHub mark from christophermanning/gist
  • Skateboard Mode: Inspired by Skatetocat from Octodex
  • Racing Mode: Custom F1 car with official octocat (scaled 35%)
  • Victory Mode: Official octocat with widened eyes for excitement

Animation Timing

  • Walking: 0.6s bounce, gentle tentacle wave
  • Skateboard: 0.4s tilt, 0.3s wheel spin
  • Racing: 0.15s vibrate, 0.4s speed lines, 0.2s wheel spin
  • Victory: 0.6s jump/rotate, 0.8s star twinkle

Size Recommendations

  • Minimum: 60px - Details still visible
  • Optimal: 100-120px - Best balance of visibility and unobtrusiveness
  • Maximum: 150px - Good for testing, may be too large for production

Implementation Notes

  • Using exact official GitHub octocat SVG paths ensures authentic look
  • All animations use CSS transforms for GPU acceleration
  • Inline SVG keeps everything in one file, no external dependencies
  • Total SVG overhead: ~8KB (acceptable for the visual impact)
  • Compatible with all modern browsers

Mode Transition Points (60s cycle)

  • 0-33% (0-20s): Walking
  • 34-66% (20-40s): Skateboard
  • 67-99% (40-60s): Racing
  • 100% (60s): Victory celebration
